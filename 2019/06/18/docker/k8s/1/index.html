IP地址 节点角色 Hostname
172.24.2.69 master k8s-master
172.24.2.71 worker k8s-node-1
172.24.2.70 worker k8s-node-2

环境准备

1、设置主机名hostname，管理节点设置主机名为 k8s-master 。

2、关闭防火墙、selinux和swap

3、配置内核参数，将桥接的IPv4流量传递到iptables的链

4、配置国内yum源

5、配置国内Kubernetes源

6、配置 docker 源
