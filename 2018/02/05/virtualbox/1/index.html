VirtualBox共享目录，建立软链接

  1. 关闭 VirtualBox。

  2. 将VirtualBox安装目录的路径加入系统环境变量PATH中。

  3. 打开命令行窗口，执行如下命令：

    参数说明
    YOURVMNAME：为虚拟机中ubuntu系统的名
    YOURSHAREFOLDERNAME：为共享的目录名称

  4. “以管理者身份运行” VirtualBox　即可！
